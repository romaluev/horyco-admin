## WebSocket Events

Subscribe to real-time inventory updates:

```javascript
// Connect to WebSocket
const socket = io('/inventory', {
  auth: { token: 'JWT_TOKEN' }
});

// Subscribe to events
socket.on('stock.low', (data) => {
  // { warehouseId, itemId, itemName, quantity, minLevel }
});

socket.on('stock.out', (data) => {
  // { warehouseId, itemId, itemName }
});

socket.on('movement.created', (data) => {
  // { movementId, warehouseId, itemId, type, quantity }
});

socket.on('po.received', (data) => {
  // { poId, poNumber, supplierId }
});
```

## Error Codes

| Code | Message                              | Description                    |
| ---- | ------------------------------------ | ------------------------------ |
| 4001 | `INSUFFICIENT_STOCK`                 | Not enough stock available     |
| 4002 | `ITEM_NOT_FOUND`                     | Inventory item not found       |
| 4003 | `WAREHOUSE_NOT_FOUND`                | Warehouse not found            |
| 4004 | `RECIPE_NOT_FOUND`                   | Recipe not found               |
| 4005 | `SUPPLIER_NOT_FOUND`                 | Supplier not found             |
| 4006 | `PO_NOT_EDITABLE`                    | PO already sent/received       |
| 4007 | `WRITEOFF_ALREADY_PROCESSED`         | Writeoff already approved      |
| 4008 | `COUNT_IN_PROGRESS`                  | Active count exists            |
| 4009 | `APPROVAL_REQUIRED`                  | Operation requires approval    |
| 4010 | `DUPLICATE_ITEM`                     | Item already exists in list    |
